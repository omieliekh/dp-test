Used technologies

  • AngularJS
  • Angular UI Router
  • RequireJS
  • NPM
  • Grunt
  • LESS
  • Highcharts-ng and Highcharts

Tricks

To make AMD approach really work, we need to trigger module loading exactly before showing its page.

Also none of module's files should be loaded before that.

So, I've implemented advanced route processing.

Angular allows us to wait for promise dependencies to be resoled

So module's files was defined as dependencies for module's route and magic happened :)

Details

For charts, highcharts-ng directive wrapper was used.

CSS was compiled with help of LESS compiler.

To simplify development, Grunt build manager was used with grunt-contrib-less to run LESS

and grunt-contrib-watch to automatically recompile CSS on every less-file change
